# Lab 05 Koncepcja komórki

## Co zostało zrealizowane
Rozwiązałem zadania analityczne z instrukcji. Obliczyłem parametry sieci komórkowych. 
Wyznaczyłem odległość wykorzystania częstotliwości. Policzyłem natężenie ruchu. Opisałem architekturę makrokomórek oraz mikrokomórek. Zawarłem odpowiedzi w pliku PDF.

## Trudności i refleksja
Rozumiem ograniczenia geometryczne przy planowaniu klastrów. 
Widzie negatywne skutki mieszania wielkości komórek w jednym systemie. 
Pilnuje jednostek czasu podczas wyliczania Erlangów. Nazywasz katalog zgodnie z narzuconym szablonem. Wgrywasz dokument na główny branch przed końcem dnia.
